Neyron Şəbəkə Arxitekturası

Layer Konfiqurasiyası

Input Layer (32×32×3 RGB)
↓
Conv2D (32 filter, 5×5) + ReLU
↓
Conv2D (32 filter, 5×5) + ReLU
↓
MaxPooling (2×2) + Dropout (25%)
↓
Conv2D (64 filter, 3×3) + ReLU
↓
Conv2D (64 filter, 3×3) + ReLU
↓
MaxPooling (2×2) + Dropout (25%)
↓
Flatten Layer
↓
Dense (256 vahid) + ReLU + Dropout (50%)
↓
Output (43 vahid) + Softmax

Arxitektura Xüsusiyyətləri

Convolutional Layer-lər ▼

Yol nişanı şəkillərindən məkan xüsusiyyətlərini və nümunələrini çıxarmaq üçün çoxsaylı filter banklarından istifadə edir. Conv2D layer-ləri şəkildən ən əhəmiyyətli xüsusiyyətləri avtomatik olaraq öyrənir və aşkar edir.

Pooling Əməliyyatları ▼

Məkan ölçülərini azaldaraq vacib xüsusiyyətləri saxlayır və hesablama səmərəliliyini artırır. MaxPooling ən güclü xüsusiyyətləri seçir və modelin ölçüsünü azaldır.

Dropout Requlyarizasiyası ▼

Təlim zamanı neyronları təsadüfi şəkildə deaktiv edərək overfitting-in qarşısını alır (25%, 50% dərəcələri). Bu, modelin ümumi qabiliyyətini artırır və yeni datalara daha yaxşı adaptasiya olunmasını təmin edir.

Dense Təsnifatlandırma ▼

Yüksək səviyyəli xüsusiyyət öyrənməsi üçün 256 vahidli tam bağlı layer və 43 sinifli softmax çıxışı. Dense layer bütün öyrənilmiş xüsusiyyətləri birləşdirərək son qərarı verir.

train.py - Əsas Təlim Dövrü
# Model Konfiqurasiyası
model.compile(
    loss='categorical_crossentropy',
    optimizer='adam',
    metrics=['accuracy']
)

# Data Artırma
datagen = ImageDataGenerator(
    rotation_range=10,
    zoom_range=0.15,
    width_shift_range=0.1,
    height_shift_range=0.1,
    shear_range=0.15,
    horizontal_flip=False
)

# Təlim İcrası
history = model.fit(
    datagen.flow(X_train, y_train, batch_size=64),
    steps_per_epoch=len(X_train) // 64,
    epochs=15,
    validation_data=(X_val, y_val)
)
Təlim Strategiyası ▼
  • Batch Size: Hər iterasiyada 64 nümunə
  • Epochs: 15-25 təlim dövrü
  • Optimizer: Adaptiv öyrənmə sürəti ilə Adam
  • Loss Function: Kateqorial çarpaz entropiya
  • Validation Split: Təlim datasının 20%-i

İzah: Bu parametrlər modelin effektiv öyrənməsini təmin edir. Batch size yaddaş istifadəsini və təlim sürətini balanslaşdırır, epochs isə modelin kifayət qədər öyrənmə vaxtı keçirməsini təmin edir.

Data Artırma Faydaları ▼
  • Rotation: Əyilmiş nişanları idarə edir
  • Zoom: Müxtəlif məsafələri simulyasiya edir
  • Shifts: Mərkəzdən kənar yerləşməni nəzərə alır
  • Shear: Baxış bucağı dəyişkənliyinə uyğunlaşır

Ümumiləşdirmə: Data artırma görünməmiş datalara ümumi qabiliyyəti artırır. Bu, real dünya şəraitində daha yaxşı performans deməkdir.

İnference və Deployment

predict.py - Real Vaxt Tanıma
model = load_model("models/traffic_sign_model.h5")
labels_df = pd.read_csv("labels.csv", header=None)
class_names = labels_df[0].str.split(" ", n=1, expand=True)[1].tolist()

def predict_sign(image_path=None, use_webcam=False):
    if use_webcam:
        cap = cv2.VideoCapture(0)
        while True:
            ret, frame = cap.read()
            img = cv2.resize(frame, (32, 32))
            img = np.expand_dims(img / 255.0, axis=0)
            
            pred = model.predict(img)
            class_id = np.argmax(pred)
            confidence = pred[0][class_id] * 100
            
            text = f"{class_names[class_id]} ({confidence:.1f}%)"
            cv2.putText(frame, text, (20, 50), 
                       cv2.FONT_HERSHEY_SIMPLEX, 1, (0,255,0), 2)
            cv2.imshow("Traffic Sign Recognition", frame)
